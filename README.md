# Wulkan
Following Vulkan Tutorial!

## Controls
| Key                 | Description            |
| ------------------- | ---------------------- |
| <kbd>W</kbd>        | Move up                |
| <kbd>A</kbd>        | Move left              |
| <kbd>S</kbd>        | Move down              |
| <kbd>D</kbd>        | Move right             |

## Quick Start

Dependencies: [vulkan], [glfw], [stb], [tinyobjloader]
```console
$ cd Wulkan/src/
$ make
$ ./Wulkan
```
